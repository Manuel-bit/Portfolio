Full Stack Web

This project in an online learning management system for Berea Christian College for Intergrated Studies (BCCIS). The web application is built on top of Django MVC framework. HTML5, CSS, JavaScript, for user views, Python for the control Logic, and PostgreSQL for the database and models.

This was my first project as a freelance web developer, The project is currently online and has facilitated online learning for more than 300 students in BCCIS through the year 2020. The project would not be a success if not for my colleague Charles Maina Murage , who played a major role in Model Definition and database design. Currently i maintain and make updates to the website in accordance to my client BCCIS specifications.
